# Versioned Example Plugin

A plugin demonstrating XClaw XClaw AgentGuard's version management system.

## Features

- Semantic versioning
- Dependency management
- Compatibility checking
- Upgrade suggestions

## Usage

```python
from xclaw_agentguard.plugins.versioned_example import VersionedExamplePlugin

plugin = VersionedExamplePlugin()
result = plugin.custom_check("test.txt", "content")
```
